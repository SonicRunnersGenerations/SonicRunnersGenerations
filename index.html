ABOUT THIS GAME

Sonic Runners Generations is a game inspired by the events of Sonic Generations, featuring the same missions, stage progression, and storyline. We've always wanted a mobile version of Sonic Generations, and this game delivers just that — offering an exciting and fun experience that combines endless running gameplay with classic stages.
